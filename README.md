This project aims to develop a non-invasive and cost-effective ML model for early detection of pancreatitis using urine sample biomarkers.
Machine Learning Model: Built and optimized models (SVM, Gradient Boosting) to classify samples into Healthy, Pancreatitis, or Pancreatic Cancer.
Multi-Class Classification: Using biomarker data to differentiate between chronic pancreatitis, early-stage, and advanced pancreatic cancer.
Web-Based Application: Developed a web interface for real-time predictions using uploaded urine data.
